# MacBook Pro (맥북프로) 제품 설명서

## 개요
맥북프로는 Apple이 설계한 프로페셔널 노트북 컴퓨터입니다. M3 Pro 또는 M3 Max 칩을 탑재하여 전문가용 워크로드에 최적화되어 있습니다.

## 주요 사양
- **프로세서**: Apple M3 Pro / M3 Max
- **메모리**: 18GB / 36GB / 48GB 통합 메모리
- **저장장치**: 512GB / 1TB / 2TB / 4TB SSD
- **디스플레이**: 14.2인치 또는 16.2인치 Liquid Retina XDR (3024×1964 / 3456×2234)
- **무게**: 14인치 1.55kg, 16인치 2.14kg

## 배터리
맥북프로의 배터리 수명은 최대 22시간(14인치 기준)으로 역대 Mac 중 가장 긴 배터리 수명을 제공합니다. 16인치 모델은 최대 24시간의 비디오 재생이 가능합니다. MagSafe 3 또는 USB-C로 고속 충전을 지원하며, 30분 충전으로 약 50%까지 충전됩니다.

## 연결성
- Thunderbolt 4 (USB-C) x 3
- HDMI 2.1 포트
- SDXC 카드 슬롯
- MagSafe 3 충전 포트
- Wi-Fi 6E, Bluetooth 5.3

## 가격
- M3 Pro 14인치: 2,390,000원부터
- M3 Max 16인치: 3,990,000원부터

## 보증
Apple 제한 보증 1년 포함. AppleCare+ 구매 시 3년으로 연장 가능합니다.
